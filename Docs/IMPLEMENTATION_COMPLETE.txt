 HOW IT WORKS
================================================================================

1. USB Inserted
2. Detect Active Window (title, process, URL, domain)
3. Find Matching Credentials (intelligent scoring)
4. Check Policy (Auto/Prompt/Never + restrictions)
5. Show Prompt (if policy = Prompt)
6. User Clicks Yes
7. Auto-Type Credentials (SendInput API)
8. Logged In!

 INTEGRATION (3 STEPS)
================================================================================

1. Register Services:
   services.AddUsbAutoInject(dataDirectory);
   services.AddSingleton<IActiveWindowDetector, WindowsActiveWindowDetector>();
   services.AddSingleton<IAutoTypeService, WindowsAutoTypeService>();

2. Wire Up Events:
   _autoInjectService.PromptRequired += OnPromptRequired;
   await _autoInjectService.StartAsync();

3. Handle Prompt:
   Show AutoInjectPromptWindow, get user choice, call AutoFillAsync()
